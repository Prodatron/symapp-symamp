1000-Introduction


INTRODUCTION TO SYMAMP

SymAmp is a music player for SymbOS, which currently supports compiled CPC Starkos modules, CPC Soundtrakker 128 modules, ZX Spectrum Protracker modules, Vortex Tracker 6channel modules, MP3 music files and Amiga MOD songs.

SymAmp provides a playlist with a full featured editor. Songs can also be played in random order. When the end of a song has been reached, the next one will be loaded and played automatically.

1300-What's new?


WHAT'S NEW?

V4.1 (2025-01-11)

- PT3 playback bug fixed
- Sound Daemon co-operation; shares sound hardware without conflicts and synchronizes master volume
- support for compressed sound files (PT3, SA2, ST2, SKM)

V4.0 (2022-01-24)

- Amiga MOD 4channel file support on platforms with an OPL4 wavetable sound device (Moonsound, MonsterSound, OPL4 Shockwave, Dal So Ri)
- new more modern skin
- enhanced playlist with up to 250 entries
- playlist accepts files from multiple folders and is sortable
- player shows full song titles if available
- correct playtime display for very long modules and MP3s

V3.0 (2021-12-15)

New driver system for all possible combinations of platforms, music hardware and sound modules. You can now play 4 different types of modules on 4 different types of Z80 platforms using 7 different types of sound hardware.
There is only one single SymAmp version again for all platforms, which can handle all combinations.

2100-Keyboard shortcuts


KEYBOARD SHORTCUTS

The following is a list of all available keyboard shortcuts.

Song control

Z jump to previous song
X play current song
C pause
V stop playing
B jump to next song
/ fast forward/backward

General

/ increase/decrease volume
L load new song
R repeat on/off
S shuffle on/off
Alt+E switch playlist on/off

Playlist editor

A add song to the playlist
Alt+A add folder to the playlist
Del remove selected songs
Clr remove all songs
Ctrl+A select all songs
Ctrl+N unselect all songs
Alt+I invert selection
U move selected songs down
D move selected songs up
Ctrl+O load new playlist
Ctrl+S save current playlist
